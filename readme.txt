How to play the game

KeyBoard    Controller
  WASD   - Left JoyStick  Move/Scroll thorugh options 
   F     -      X         Interact
   J     -      B         Spectral Dive
SpaceBar -      A         Spectral Jump
   K     -      Y         Spectral Host
  ESC    -    Start       Pause Game
 Enter   -      A         Select Option
